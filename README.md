# Hacker News

Interface for Hacker News website

## Technologies stack

This project was bootstrapped with create vite@latest.

- React
- React Router V6
- React Redux
- Matirial UI
- Npm Package manager

## API

Official [Hacker News API](https://github.com/HackerNews/API)

## Setup

- clone repo
- cd HN-project
- npm install
- npm run dev
- and browser open by default to http://localhost:3000
